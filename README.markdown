# Simple Javascript "add" library

A Javascript package that sum a sequence of integer numbers.

## Install

```
npm install
```

## Build

```
npm run build
```

## Run

### Lint

Run the lint.

```
npm run lint
```

### Test

Run the test suite.

```
npm run test
```

### Output example

```
$ npm run test
> add@1.0.0 test C:\Users\foo\TypeScript\add
> alsatian "./test/**/*.spec.ts"


 Pass: 5 / 5
 Fail: 0 / 5
 Ignore: 0 / 5
```
